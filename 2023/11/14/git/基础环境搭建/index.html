利用Github Pages&Hexo 搭建博客系统

Github Pages设置
  • 在Github中创建一个新的Repository，命名为 <username>.github.io,<username>为Github的用户名
  • 使用git链接到对应Repository，新建一个分支用于存放静态文件
  • 后续同步修改可以通过git直接同步到远程仓库
npm环境设置
  • 当前在Archlinux下搭建npm环境
Hexo环境设置
  • 完成Hexo安装后，在主机新建Hexo博客项目
  • 配置Hexo博客的基本信息，修改hexo目录下的_config.yml文件，按需配置
  • 生成静态页面
  • 生成完成后，使用命令启动本地服务
  • 服务启动后，通过 http://localhost:4000 即可预览博客页面
Hexo发布服务
  • 安装hexo-deployer-git插件
  • 利用Hexo d直接上传到github
  • 打开<username>.github.io即可
    此间可能会出现github分支无法合并的问题，请百度
Hexo主题应用

  • 搜索hexo的主题，在hexo init目录下找到theme文件夹，切换到对应文件后git clone该主题仓库

  • 修改主题仓库中的_config.yaml文件，参考对应主题项目的readme.md

  • 同样在hexo init目录下，修改_config.yml文件，将theme的参数设置为git clone的文件夹名字

  • 再次hexo clean hexo g hexo d 刷新缓存，生成静态页面，同步到github
